Introduction to Sets

A set is a collection of objects. Each object in a set is called an element. We can define a set by listing out its elements:

A = {a1, a2, a3, ...}

For example, the set of natural numbers would be:

ℕ = {1, 2, 3, 4, ...}

And the set of integers would be:

ℤ = {..., -2, -1, 0, 1, 2, ...}

To denote that a1 is an element of a set A, we write:

a1 ∈ A

For example, to denote that 245 is an element of the natural numbers, we would write:

245 ∈ ℕ

Some sets are completely included inside of other sets. For example, suppose A = {a, b, c} and B = {a, b, c, d, e}. Then we would say that A is a subset of B and we would write:

A ⊂ B

For example, to denote that the natural numbers are a subset of the integers, we would write:

ℕ ⊂ ℤ

Common Sets of Numbers

There are some sets of numbers that we use often, such as natural numbers and integers. These sets have been assigned special names and symbols.

Name Symbol Description Elements
Natural Numbers ℕ The counting numbers starting at 1 1, 2, 3, 4, 5, 6, ...
Integers ℤ The natural numbers plus the negative whole numbers and zero ..., -2, -1, 0, 1, 2, ...
Rational Numbers ℚ The integers plus fractions examples: 1, -1/2, 17/54
Real Numbers ℝ The rational numbers plus irrational numbers
(numbers that can't be written as fractions)		 examples: 1, -1/2, √2, π

You might have noticed that the natural numbers are a subset of the integers. Similarly, the integers are a subset of the rational numbers, and the rational numbers are a subset of the real numbers. We can write this all out as:

ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

Set-Builder Notation

Sometimes it is impractical to list out all of the numbers in a set. In this case, often we will use a special notation called set-builder notation. This notation defines the set by describing the properties of the numbers that are elements of the set. See the following example:

{x | -4 < x < 7, x ∈ ℤ}

Note: This means "the set of numbers x such that x is between -4 and 7 and x is an integer."

If we were to list out all the elements of this set, it would look like this:

{-3, -2, -1, 0, 1, 2, 3, 4, 5, 6}

Here is another example, showing how we could write "the set of all rational numbers between 1/2 and 50":

{x | 1/2 < x < 50, x ∈ ℚ}

We can even use set-builder notation to represent more complicated sets, such as the even numbers:

{x | x = 2n, n ∈ ℤ}

Intervals

One type of set that we use very often is the interval. An interval is the set of all real numbers between two endpoints. For example, if we wanted the set of all real numbers greater than 1 and less than or equal to 15, we could write:

{x | 1 < x ≤ 15}

Note: If we do not indicate what type of number x is, it is just assumed to be a real number.

Since we use intervals so often, we have a special way to write them called interval notation. We could write the set above using interval notation in this way:

(1, 15]

Note: The round bracket ( means "don't include 1". The square bracket ] means "do include 15".

See the table below for a complete list of different types of intervals:

Bounded Intervals Unbounded Intervals
Set-Builder Notation Interval Notation Set-Builder Notation Interval Notation
{x | a < x < b} (a, b) {x | x < a} (-∞, a)
{x | a ≤ x ≤ b} [a, b] {x | x > a} (a, ∞)
{x | a < x ≤ b} (a, b] {x | x ≤ a} (-∞, a]
{x | a ≤ x < b} [a, b) {x | x ≥ a} [a, ∞)
